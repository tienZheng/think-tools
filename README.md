<h1 align="center"> think-tools </h1>

<p align="center"> ThinkPhp5 常用工具.</p>

## 项目简介

1. 封装了加工请求状态码类，状态码常量化，更具语义。
2. 封装了数组及字符串常用的几个处理函数。


## 安装

```shell
$ composer require tien/think-tools
```
